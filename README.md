# Recipin – Recipe Bookmark Chrome Extension

**Recipin** is a lightweight Chrome extension that lets you save your favorite recipe links with just a name. Quickly store and access recipes without the need for accounts or complicated setups.

## Features

- **Add Recipe Name and Link:** Simply enter the name of the recipe and press **Save**.  
- **Local Storage:** All recipes are saved in your browser’s local storage.  
- **Quick Access:** Open your saved recipes anytime from the extension.  
- **Minimal & Simple:** No unnecessary features – just save and view recipes.
